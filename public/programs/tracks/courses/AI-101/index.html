AI-101 AI for Everyone and Fundamentals of Programming using Python

Description

We will start this course by understand the fundamentals and use cases for AI. Then, we'll learn about basic programming concepts using Python, such as lists, dictionaries, classes, functions and loops, and practice writing clean and readable code with exercises for each topic. We'll also learn how to make your programs interactive and how to test your code safely before adding it to a project. It is a fast-paced, thorough introduction to programming with Python 3.8 that will have you writing programs, solving problems, and making things that work in no time.
